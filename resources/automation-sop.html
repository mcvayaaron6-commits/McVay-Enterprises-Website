Automation SOP

Faith Commerce Automation Blueprint

Connect Shopify, AutoDS, Printify, and ChatGPT so every order is fulfilled on autopilot.

Pre-Launch Checklist

Complete these items before activating automations:

  • ✅ Create a Shopify store and install the Dawn theme.
  • ✅ Upgrade AutoDS to the "Advanced" plan for automated ordering.
  • ✅ Connect Printify and enable your top POD suppliers (Monster Digital, SwiftPOD, printHQ).
  • ✅ Verify your business email domain so branded notifications are not flagged as spam.
  • ✅ Add a Shopify staff account for automation@mcvayenterprises.com to monitor flows.

System Connections

1. Shopify ↔ AutoDS

Install the AutoDS app, connect your store, and enable Fulfilled by AutoDS. Set default profit rules to cost × 2.4 + shipping and enable automatic price updates.

2. AutoDS ↔ Ultimate Shield

Inside AutoDS, choose Ultimate Shield as the preferred supplier for bracelet SKUs. Toggle "Auto order" and upload your branding insert under Custom Thank You Card.

3. Shopify Flow

Create a Flow automation: Trigger → Order paid, Condition → Tag contains chatgpt, Action → Send internal email with order summary and supplier SKU.

4. Klaviyo Integration

Connect Klaviyo via API keys. Sync all customer profiles and enable the prebuilt ChatGPT drop welcome series (7 emails, 2 SMS).

Automation Timeline

Day 1 - 2: Fulfillment Foundation

  • Install AutoDS, DSers, and Syncee for multi-supplier coverage.
  • Configure profit automation and shipping policies (US + International).
  • Map product variants to suppliers and import top six SKUs.

Day 3 - 4: ChatGPT Commerce

  • Create the Faith Commerce Assistant GPT with Shopify Actions enabled.
  • Sync product titles, prices, and images to the GPT knowledge base.
  • Test the end-to-end flow: ask for a bracelet suggestion → add to cart → confirm order appears in Shopify.

Day 5 - 7: Marketing Automation

  • Launch Klaviyo welcome + cart recovery sequences using the provided templates.
  • Connect Zapier: New order (Shopify) → Add customer to Airtable CRM → Slack notification.
  • Schedule TikTok & Instagram reels with Later.com, auto-tagging SKUs in captions.

Quality Assurance

Weekly Automation Audit

  • Run a test order through ChatGPT to confirm the checkout link is generated.
  • Verify AutoDS sends supplier tracking numbers back to Shopify within 24 hours.
  • Ensure Klaviyo flows are triggered (check analytics for open and click rates).
  • Review Zendesk/Tidio transcripts for unanswered automation questions.

Support & Escalation

Have a manual fallback ready:

  • orders@mcvayenterprises.com receives all failed automation alerts.
  • If AutoDS is down, duplicate the order inside Shopify and forward it directly to the supplier via CSV.
  • Log all incidents in the "Automation QA" tab inside the KPI dashboard template.
