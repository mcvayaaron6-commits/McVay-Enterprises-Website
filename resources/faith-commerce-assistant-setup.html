Faith Commerce Assistant GPT

Follow these steps to launch the ChatGPT assistant that recommends products, generates checkout links, and captures leads for McVay Enterprises.

1. Create the GPT

  • Open ChatGPT Builder and click Create a GPT.
  • Name it Faith Commerce Assistant and add a description such as "Your Christian gift concierge powered by McVay Enterprises."
  • Upload the product sheets from /resources/products and the bundle playbook to the knowledge base.

2. Configure Actions

Shopify Checkout Links

  • Create a Shopify custom app with access to Draft Orders API.
  • Use the provided checkoutLink endpoint (stored in your infrastructure) to generate draft checkouts based on product handles.
  • Connect the API inside the GPT Actions tab with the OpenAPI schema below.
{
  "openapi": "3.1.0",
  "info": { "title": "Faith Commerce Checkout", "version": "1.0.0" },
  "servers": [{ "url": "https://api.mcvayenterprises.com" }],
  "paths": {
    "/checkout": {
      "post": {
        "operationId": "createCheckout",
        "summary": "Create Shopify draft checkout",
        "requestBody": {
          "required": true,
          "content": {
            "application/json": {
              "schema": {
                "type": "object",
                "properties": {
                  "items": {
                    "type": "array",
                    "items": {
                      "type": "object",
                      "properties": {
                        "handle": { "type": "string" },
                        "quantity": { "type": "integer" }
                      },
                      "required": ["handle", "quantity"]
                    }
                  },
                  "email": { "type": "string" }
                },
                "required": ["items"]
              }
            }
          }
        },
        "responses": {
          "200": {
            "description": "Checkout link created",
            "content": {
              "application/json": {
                "schema": {
                  "type": "object",
                  "properties": {
                    "checkoutUrl": { "type": "string" }
                  },
                  "required": ["checkoutUrl"]
                }
              }
            }
          }
        }
      }
    }
  }
}

3. Conversation Prompts

  • Use the Creative Prompts resource to seed welcome messages and upsell logic.
  • Define guardrails: stay within Christian merchandise, offer bundle suggestions, capture email for follow-up.
  • Add fallback response linking to the Shopify store if the API call fails.

4. Publish & Embed

  • Publish the GPT and copy the shareable link.
  • Update config.chatgptStoreUrl in main.js with the new link so all CTA buttons open the assistant.
  • Announce the GPT via newsletter and add it to the ChatGPT store tags: ecommerce, faith, gift-guide.

5. Tracking

  • Add a query parameter (e.g., ?utm_source=chatgpt) to generated checkout links for analytics.
  • Tag all GPT-generated orders in Shopify Flow for reporting inside the KPI dashboard.
  • Review usage metrics weekly to optimize prompts and recommended bundles.
